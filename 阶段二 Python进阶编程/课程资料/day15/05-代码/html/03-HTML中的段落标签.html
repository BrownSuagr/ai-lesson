俄外交部副部长透露：普京可能在今年年底前同拜登会面

【环球网报道】据“今日俄罗斯”（RT）报道，俄罗斯外交部副部长里亚布科夫当地时间21日透露，俄罗斯总统普京可能在今年年底前同美国总统拜登会面。RT称，这将是两位领导人在2021年里的第二次面对面接触。

报道称，里亚布科夫当天告诉俄媒，组织这次活动的工作已经在进行中。这位俄高级外交官表示，华盛顿最近发表了一些与俄罗斯相关的言论，包括那些与乌克兰局势有关的内容，莫斯科认为有必要“详细解释到底发生了什么，以及是如何发生的”。

“进行这次会面需要做充分的准备，这就是我们正在做的。”里亚布科夫说。

报道提到，本周早些时候，克里姆林宫发言人德米特里佩斯科夫也提到了普京在不久的将来与拜登会面的可能性。然而，他表示，尚未就具体时间表达成一致。当地时间11月18日接受俄媒采访时，佩斯科夫称，“（会面）有没有可能在新年之前？是的，有这个可能。”不过他也补充说，“也可能是在之后”，这最终取决于两位领导人的工作安排。
